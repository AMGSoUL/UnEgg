UnEGG linux version(v0.5)
---------------------------

1. General
 This package includes a source code of UnEGG commandline software ("UnEGG") and makefile.
 The UnEGG is a freeware that allows to decompress EGG and ALZ formats and view list.
 The UnEGG includes the source of decompression algorithm. Please refer to the license below.

2. Limitation
 The UnEGG source code was tested in Ubuntu (11.04) and Mac OSX (10.6.x).
 The UnEGG does not support for Big Endian. This will be supported in the future.
